# Features

Reusable feature modules. I divided these up based on how likely I would personally enable/disable them on each of my machines.
This is entirely subjective and just based on 'feel'.

## Description

| Feature        | Description                                                                 |
| -------------- | --------------------------------------------------------------------------- |
| `base`         | The base config applied to all hosts.                                       |
| `desktop`      | For workstation hosts with a screen (laptop or desktop).                    |
| `drivers`      | A collection of hardware-related modules; enable/disable per host hardware. |
| `gaming`       | Configuration and optimizations for gaming programs.                        |
| `self-hosting` | Tooling for hosting services.                                               |
